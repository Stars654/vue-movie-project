# 电影评价与推荐系统

## 项目简介
这是一个基于Vue.js和Node.js的电影评价与推荐系统，用户可以对电影进行评价、评分和推荐。

## 功能特性
- 用户注册和登录
- 电影推荐和查看
- 电影评分和评论
- 电影搜索和筛选
- 热门电影推荐
- 后台管理系统

## 技术栈
- 前端：Vue.js 3 + Element Plus + Axios
- 后端：Node.js + Express + MySQL
- 数据库：MySQL
